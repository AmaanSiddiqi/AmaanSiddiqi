## 🔨 Things I've built

### 🧬 Colony — Neural Darwinism Agent System
Multi-agent LLM framework where agents evolve through competitive selection and Hebbian learning. No hard-coded roles — specialization emerges. Includes a two-tier memory system with ChromaDB RAG + live LoRA fine-tuning that hot-swaps adapters mid-inference.

### 📊 Vancouver Job Analyzer — [**live ↗**](https://jobs.amaansiddiqi.me)
Automated scraper that tracks tech job postings across Vancouver every 6 hours, extracts skills via spaCy NLP, and surfaces week-over-week demand trends in a React dashboard. Actually useful if you're job hunting in Vancouver.

### 🌲 Korotu — LiDAR Tree Species Classification
End-to-end ML pipeline for classifying tree species from drone-captured LiDAR point clouds (10M+ points). Hit **80% genus / 70% species accuracy** using hierarchical PointNet++. Presented at UBC Design & Innovation Day.

### 🏃 Somatic Syntax — Real-Time Posture Analysis
Microservices backend + GPU inference pipeline for real-time pose analysis at 17–20 FPS. YOLOv8-Pose + MMPose, fully containerized, with TimescaleDB for high-frequency time-series storage.
